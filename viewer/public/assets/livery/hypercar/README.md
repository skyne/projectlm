# Legacy hypercar line-art assets (deprecated)

These PNGs (`*-outline.png`, `*-mask.png`, `*-details.png`, `sheet.png`) were an earlier livery pipeline. **The viewer no longer uses them.**

Livery previews now render through the compositor (`liveryRenderer.ts` → `defaultLiveryVisual` / `pickHypercarChassis`). Files are kept for reference only and may be removed later.
